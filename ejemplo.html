Cómo crear tus propios archivos HTML

Esta página es una guía simple para que puedas crear y publicar tus propias páginas web, incluso si no tienes mucha experiencia con tecnología. Sigue los pasos en orden.

Paso 1: Entender la estructura básica de un HTML

Todo archivo HTML tiene esta estructura mínima. Cópiala y pégala como punto de partida:

<!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="UTF-8">
  <title>Título de mi página</title>
</head>
<body>
  <h1>Mi primer título</h1>
  <p>Este es un párrafo de texto.</p>
</body>
</html>

Paso 2: Crear tu archivo

  1. Dentro de la carpeta proyecto-web, crea un archivo nuevo. Por ejemplo: mi-pagina.html
  2. Pega la estructura del Paso 1 dentro de ese archivo.
  3. Cambia el texto entre <h1> y </h1> por el título que quieras.
  4. Agrega más párrafos con la etiqueta <p>...</p>

Paso 3: Etiquetas básicas más usadas

  • <h1> a <h6> → Títulos (h1 es el más grande)
  • <p> → Párrafo de texto
  • <img src="ruta-de-la-imagen.jpg"> → Insertar una imagen
  • <a href="https://ejemplo.com">Texto del enlace</a> → Enlace a otra página
  • <ul><li>...</li></ul> → Lista con viñetas
  • <table> → Tabla

Paso 4: Ver tu página en el navegador

Esta VM ya tiene un servidor activo que muestra los archivos de la carpeta proyecto-web en internet. Cualquier archivo .html que crees ahí se podrá ver desde tu navegador.

Por ejemplo, este mismo archivo se puede ver en:
https://nbsilguero-cpu.github.io/omi/ejemplo.html

Paso 5: Publicar tus cambios (subir a internet)

Cada vez que crees o modifiques un archivo, pídele a tu asistente que ejecute estos 3 comandos (o ejecútalos tú en la terminal, dentro de la carpeta proyecto-web):

git add .
git commit -m "Agrego mi nueva página"
git push

Después de unos minutos, tu página estará disponible en internet en la dirección de GitHub Pages.

Consejo: No te preocupes si algo se ve mal al principio. Guarda el archivo, recarga el navegador (tecla F5) y revisa los cambios. Puedes repetir esto las veces que quieras.

Paso 6: Personalizar con estilos (CSS)

Para cambiar colores, tamaños y espacios, se usa CSS. Puedes agregar estilos directamente dentro de tu archivo HTML, dentro de una etiqueta <style> en el <head>, como se hizo en esta misma página. 

<style>
  h1 {
    color: blue;
  }
</style>
